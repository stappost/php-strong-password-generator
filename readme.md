## Scaffoldin
1- creare form che permetta di inserire la lunghezza della password
2- creare variabile con all'interno tutti i caratteri
3- ciclare i caratteri in base alla lunghezza desiderata
4- inserire casualmente i caratteri in una nuova variabile
5- trasformare la nuova variabile in una stringa
6- restituire la password generata all'utente
7- spostare la logica in un nuovo file
8- includiamolo in index
9- creiamo un altro file per mostrare la password
10- recuperare la password nel file tramite $_SESSION
11- effettuare un redirect al file
12- mostriamo la password
13- permette all'utente di scegliere quali caratteri utilizzare
